# Incident Response

This directory contains incident response reports developed as part of the Google Cybersecurity Professional Certificate.

The reports apply the **NIST Cybersecurity Framework (CSF)** to real-world security scenarios, focusing on structured analysis, response actions, and post-incident improvements.

## Contents

- **ICMP Flood Denial of Service (DoS)**
  - Analysis of a network-based DoS attack caused by ICMP flooding
  - Root cause identification (firewall misconfiguration)
  - Application of NIST CSF functions:
    - Identify
    - Protect
    - Detect
    - Respond
    - Recover

## Key Concepts & Tools

- NIST Cybersecurity Framework (CSF)
- Network Security
- Firewall Configuration
- IDS/IPS
- Network Monitoring
- Incident Analysis

## Purpose

The goal of this documentation is to demonstrate a structured and practical approach to incident response, emphasizing clarity, technical reasoning, and alignment with industry-recognized security frameworks.
